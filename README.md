# GYOF- Grow your own food

OPENAG-India.
Based on OpenAG initiative by MIT.

** ##Sensors **
1. AM2315
2. SI7021
3. pH sensor
4. Electrical Conductivity sensor
5. DS18B20 
6. DHT22
7. Optomax digital liquid sensor

** ##Relays **
1. White LEDs (AC)
2. LED Panel (24V DC)
3. Circulation Fan (12V DC)
4. Exhaust Fan (12V DC)
5. Air pump (AC)
